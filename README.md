# DeepLearning-with-TensorFlow-Keras

A hands-on repository for learning core **TensorFlow fundamentals** through concise explanations and practical Jupyter Notebook implementations.

## 📚 Topics Covered

* Tensor creation, data types, attributes, indexing, and shapes
* Tensor manipulation, mathematical operations, and matrix multiplication
* Aggregation and positional Min/Max operations
* Random tensors and shuffling
* NumPy and TensorFlow interoperability
* One-hot encoding and GPU availability
* Automatic differentiation using `tf.GradientTape()`
* Gradient computation and Gradient Descent
